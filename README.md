# RSA Accumulator-based-CLPKS Scheme
> A prototype of Certificate Less Public Key Signature based on RSA Accumulator Scheme.

## Environment
- **OS**: Ubuntu22.04 (WSL2)
- **Build**: CMake
- **Architecture**: x86_64
- **Dependencies**:
    - MIRACL Cryptography Library
    - jsoncpp
